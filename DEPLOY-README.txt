  PRESENCE LAYER · ХОСТИНГ ПРОТОТИПА
  Бесплатно. Постоянная ссылка. Открывается с телефона.
══════════════════════════════════════════════════════════

ВАРИАНТ 1 · ОДИН КЛИК (рекомендую)  ─────────────────────
  1. Дважды кликни на файл  deploy.bat  в этой папке
  2. При первом запуске спросит:
       email:    <твой email>
       password: <придумай любой, запомни>
  3. Через 30 секунд получишь ссылку:
       https://presence-layer.surge.sh

  Эту ссылку можно отправлять кому угодно — открывается
  и с компа, и с телефона.


ВАРИАНТ 2 · NETLIFY DROP (через браузер, без терминала) ─
  1. Открой:  https://app.netlify.com/drop
  2. Залогинься через Google (5 секунд)
  3. Перетащи ВСЮ эту папку (presence-prototype) на страницу
  4. Получишь ссылку вида:  https://random-name.netlify.app
  5. (опц.) В Site settings → Change site name → presence-layer


ВАРИАНТ 3 · VERCEL (тоже через браузер)  ────────────────
  1. https://vercel.com/new
  2. Login → Google
  3. "Deploy without Git" → перетащи папку
  4. Получишь ссылку  https://presence-layer.vercel.app
